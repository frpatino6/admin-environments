⚠️  IMPORTANTE: Tu versión actual de Node.js es v12.22.12
    Se requiere Node.js >= v20.19 o v22.x

1️⃣  ACTUALIZAR NODE.JS
    → Descargar desde: https://nodejs.org/
    → Instalar versión LTS (v22.x)
    → Reiniciar VS Code

2️⃣  INSTALAR DEPENDENCIAS
    cd backend && npm install
    cd ../frontend && npm install

3️⃣  CONFIGURAR SLACK
    → Seguir guía en SLACK_SETUP.md
    → Editar backend/.env con tu webhook URL

4️⃣  INICIAR APLICACIÓN
    
    Opción A - PowerShell:
    .\start.ps1
    
    Opción B - Manual:
    cd backend && npm run dev     (Terminal 1)
    cd frontend && npm start      (Terminal 2)
    
    Opción C - VS Code:
    Ctrl+Shift+P → Tasks: Run Task → Start All

5️⃣  INICIALIZAR BASE DE DATOS
    curl -X POST http://localhost:3000/api/environments/init

6️⃣  ABRIR APLICACIÓN
    → Frontend: http://localhost:4200
    → Backend:  http://localhost:3000
